State Messaging

"com.kinstrife.core.state-messaging": "file://GameDev/Core/Packages/StateMessaging" Package that builds on Messaging, & Stateflow and provides implementation utilities for messaging state-based types, and coupling with flowgraphs. Additionally this package is home to core channel handles that use the StateID. message type

Channels

  • PrimaryModuleChannelHandle
  • UIChannelHandle

Behaviours

The behaviours get a flowgraph injected, and will couple with them in a given direction.

Abstract

The following abstract MonoBehaviours are available to extend (check out the concrete implementations on how-to):

  • StatePublisherBehaviour<TMessage> - Publishes over a selected channel when the injected FlowGraphBase changes state fires the OnNewState event
  • StateSubscriberBehaviour<TMessage> - Subscribes to a selected channel, and applies the received state message to the state machine
