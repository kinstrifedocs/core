To create a state machine:

  • Create a FlowGraph instance, which will contain the state machine flow graph

  • Create StateID instances, one for each unique state you want in your state machine.

    • Names (which will be displayed in the graph) will be useful to document the intent of the states.

  • Open the FlowGraph by double clicking it

  • Create FlowNodes by right clicking on the canvas
