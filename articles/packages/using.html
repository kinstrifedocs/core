Using packages

 

Packages can either be added directly to a project, or added as a dependency of other packages.

 

To add a package to a project, open its manifest.json (located in the project's Package folder). At the beginning of the file, add:

"com.kinstrife.packagename": "file:/GameDev/Core/Packages/PackageName",

 

To add a package as a dependency to another package, open its package.json (located in the other package's root folder). In the dependencies object in the .json file, add the following line:

"com.kinstrife.packagename": "file:/GameDev/Core/Packages/PackageName#1.0.0",

 

Note

You can find the package identifier and relative path on the documentation pages of the package. They are located right below the title and can simply be copy and pasted. When copying for a package dependency, you must manually add the SemVer tag #1.0.0 as in the example above.
