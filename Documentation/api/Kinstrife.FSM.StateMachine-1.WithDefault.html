The starting state and main state to return to when nothing else is active.

Declaration
public TState DefaultState { get; set; }
Property Value
Type Description
TState
Remarks

If the CurrentState is null when setting this value, it calls ForceSetState(TState) to enter the specified state immediately.

For a character, this would typically be their Idle state.
