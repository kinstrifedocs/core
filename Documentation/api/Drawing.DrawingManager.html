Class DrawingManager

Global script which draws debug items and gizmos. If a Draw.* method has been used or if any script inheriting from the MonoBehaviourGizmos class is in the scene then an instance of this script will be created and put on a hidden GameObject.

It will inject drawing logic into any cameras that are rendered.

Usually you never have to interact with this class.

GetRedrawScope()

A scope which can be used to draw things over multiple frames. You can use GetBuilder(RedrawScope, Boolean) to get a builder with a given redraw scope. After you have disposed the builder you may call Draw() in any number of future frames to render the command builder again.

Note: The data will only be kept if Draw() is called every frame. The command builder's data will be cleared if you do not call Draw() in a future frame. After that point calling Draw() will not do anything.
