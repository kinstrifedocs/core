AudioGroup

Group used for limiting number of sound instances (e.g. footsteps) throughout the game world This is useful as a spatially aware alternative to Wwise's max sound count feature for retaining 3D information (e.g. to allow sounds to play in front of the player and behind them, whereas otherwise all sounds behind the player might be culled)
