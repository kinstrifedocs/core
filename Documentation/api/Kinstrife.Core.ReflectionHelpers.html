Namespace Kinstrife.Core.ReflectionHelpers

Classes

ExtendedTypeInfo

Contains additional utilities for a type, such as getting member values based on attributes and an optimized way to read/write to fields/properties.

GenericMethodInfoHelper

Help construct non-generic System.Reflection.MethodInfo from generic methods that can be used to invoke them. This is useful when a generic method needs to be invoked, but only a Type reference is available, not a Type symbol, as is often the case when using reflection. Only works for methods with a single generic type parameter.

TypeAccessor

Access fields and properties in an optimized manner. Utilizes for properties, but as FastMember only supports properties, this wrapper is needed to also offer this functionality for fields.

TypeHelpers

Various utilities for using Types with reflection.

TypeInfoCache

Access extended type info such as attribute data and access instance members in an optimized manner.
