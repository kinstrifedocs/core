Type Parameters
Name Description
TState
Remarks

This class doesn't keep track of any states other than the currently active one. See StateMachine<TKey, TState> for a system that allows states to be pre-registered and accessed using a separate key.

See InitializeAfterDeserialize() if using this class in a serialized field.
