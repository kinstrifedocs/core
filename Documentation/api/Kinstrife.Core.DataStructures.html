Namespace Kinstrife.Core.DataStructures

Classes

BiDictionary<TAlpha, TBeta>

Stores one-to-one correspondence (a bijection, if you want to get mathematical). This implementation only works for two different types. The reason for this is that indexing would not work if both types were the same - this is a result of choosing convenience features over flexibility.

BinaryHeap<T>

A binary heap is an implementation of a priority queue, which is a queue in which elements additionally have an assigned priority. Sorting is based on the heap's Compare method, which determines which element should be popped first.

CircularBuffer<T>

Basically a limited size queue. Not threadsafe.

Pair<T>

Generic pair structure which is particularly useful for dictionaries, e.g. for mapping connecting lines to the pairs they connect. The order of A and B does not matter when comparing pairs.
